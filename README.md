# prueba-puxles-microprofile

Herramientas usadas:
IDE: Eclipse
Backend:
  - Thorntail microprofile : [https://thorntail.io/](https://thorntail.io/)
  - Hibernate
  - MongDB
  - Java JDK 8
  - Swagger
 
Para ejecutar backend: mvn thorntail:run
  
Frontend:
  - Angular 9
  
Para ejecutar front:
- npm install (Instala dependencias)
- ng serve o npm start
